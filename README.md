# Retail Sales Power BI Dashboard

I built this project to analyze 4 years of retail sales data using Power BI.  
The goal was to understand sales trends, regional performance, and forecast future growth using time-series analysis.

## Project Highlights
- Designed KPI cards to track Sales, Profit, Quantity, and Shipping Days
- Analyzed sales performance across regions and categories
- Created interactive charts to explore monthly and yearly trends
- Implemented forecasting to predict future sales patterns

## Tools & Skills Used
Power BI, Power Query, DAX, Data Cleaning, Data Modeling
